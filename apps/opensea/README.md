# OpenSea Creatures Viewer

## Steps

* Add Infura projectId to `/apps/opensea/src/secrets.json`.
* `make start`
* `make bash`
* `cd opensea`
* `yarn install`
* `yarn run proxy`
* Open new terminal window.
* `make bash`
* `cd opensea`
* `yarn run app`
* http://localhost:5800
* Install Metamask.
* http://node:3000
