# ethereum-playground

Ethereum Playground

## Usage

* `make build`
* `make start`
* `make bash`
  * `cd {{ app }}`
  * `npm run solhint`
  * `truffle compile`
  * `truffle test`
  * `truffle migrate`
  * `lite-server`
* `make logs`
* `make stop`

## URLs

* firefox: http://localhost:5800
* ganache: http://ganache:8545
* node: http://node:3000
